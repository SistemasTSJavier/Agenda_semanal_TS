# send-reservation-email

Envía un correo de recordatorio cuando se crea una reservación con "Correo (notificación)" rellenado.

- **Gratis:** Resend — solo configurar `RESEND_API_KEY`.
- **Opcional (tu dominio):** SMTP Office 365 — `SMTP_USER`, `SMTP_PASS`, opcional `SMTP_FROM`.
- La app Flutter invoca esta función después de crear la reserva.

Ver en la raíz del proyecto: **NOTIFICACIONES-EMAIL.md**.
